# GitHub访问加速
## 修改 Hosts 文件
下载SwitchHosts这个工具定时修改host文件，下载地址：https://github.com/oldj/SwitchHosts
### 操作步骤
添加一条规则：

方案名：GitHub（可以自行命名）
类型：远程
URL 地址：https://gitlab.com/ineo6/hosts/-/raw/master/hosts
自动更新：1个小时（时间可自行调整）
这样就可以和最新的hosts保持同步。
