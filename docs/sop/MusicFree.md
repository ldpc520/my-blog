

# MusicFree音源接口汇总

## 一、 软件介绍
MusicFree是一款开源、免费、可定制化的本地音乐 播放器。

## 二、配置方法

### MusicFree PC版 配置方法

1. 点击主界面左侧【插件管理】
2. 点击顶部【订阅设置】
3. 输入插件 URL 保存，然后更新订阅即可（插件链接以 .json 或 .js 结尾）
